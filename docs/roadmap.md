# EV-Locater Hackathon Roadmap ⚡

## 🚀 Goal
Build a high-impact, smooth demo of a smart EV charging platform within a 48-hour sprint.

---

## 🛠️ Tech Stack (Hackathon Optimized)
- **Frontend**: React (Vite) + Tailwind CSS + **Framer Motion** (for the 'wow' factor).
- **Backend**: Node.js + Express + MongoDB.
- **Maps**: Leaflet (Fastest setup).
- **Images**: Cloudinary.
- **SMS**: Fast2SMS.

---

## 📅 48-Hour Sprint Plan

### 0-12h: Foundation
- Repository init & DB connection.
- Fast2SMS login integration.
- UI Design System setup (Neon Dark Mode).

### 12-24h: Geo-Map & Discovery
- Seed database with dynamic station records.
- Map search & nearby station listings.
- Image uploads via Cloudinary.

### 24-36h: Booking & Real-time
- Slot selection UI.
- Socket.io for real-time status (Show status change on map).
- "Mock" Payment flow with Razorpay.

### 36-48h: The Showstopper (Route Planner)
- Implement Route-Based charging stop suggestions.
- Final UI polish & Demo preparation.
- Deployment to Vercel/Render.
